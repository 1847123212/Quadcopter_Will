const uint32_t g_lpuartBaseAddr[HW_LPUART_INSTANCE_COUNT]

This demo initializes LPUART module for communication. The FreeMASTER driver itself supports multiple types of communication, but it requires the module to be initialized before it starts. Refer to other freemaster_demo applications in the parent directory to see how other communication modules are initialized. This demo uses the default LPUART module as it is defined in the board.h file. You can choose a different LPUART instance by rewriting in the main() function below. FreeMASTER supports two ways haw to select the communication module instance. Either statically via module base address defined as FMSTR_SCI_BASE in freemaster_cfg.h file, or dynamically through FMSTR_SetSciBaseAddress() function call. This demo uses the later case. It makes no sense to define the FMSTR_SCI_BASE in this demo.All compile-time checks passed, include LPUART HAL driver files Table of base addresses for lpuart instances.
